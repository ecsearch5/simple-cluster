Select k+1-th smallest member of an array : QuickSelect by Hoare. This one take O(n) time in average but O(n^2) in the worst case. QuickSelect might be slower than StableSelect in worst cases, but in most cases, QuickSelect outperformed StableSelect. BE CAREFUL! This function will change the order of the input data. Make sure you stored your input data somewhere else.
